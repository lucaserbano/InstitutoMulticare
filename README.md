# Instituto Multicare

Site institucional do **Instituto Multicare** (Batel, Curitiba/PR) — dermatologia com a **Dra. Adriana Matter** e ortopedia, medicina do estilo de vida e longevidade com o **Dr. Douglas Moraes**.

Site estático (HTML/CSS/JS), sem etapa de build.

## Rodar localmente

```bash
python3 -m http.server 8000
# abrir http://localhost:8000
```

## Estrutura

- `index.html` — Início
- `adriana.html` — Dra. Adriana
- `douglas.html` — Dr. Douglas
- `assets/` — `css/`, `js/`, `fonts/`, `img/`
